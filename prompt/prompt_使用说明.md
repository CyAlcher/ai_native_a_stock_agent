# A股Agent提示词模板 — 使用说明

---

## 文件结构

```
prompt/
├── 00_变量速查表.md          ← 先看这个，所有变量在这里
├── 01_空仓观望_template.md   ← 用这些
├── 02_信息搜集_template.md
├── ...
├── 11_等待下一次机会_template.md
├── 01_空仓观望.md            ← 原始文件，不用动
└── ...
```

`_template.md` 是你实际使用的文件，原始 `.md` 是备份，不需要碰。

---

## 核心概念：变量

模板里所有 `{{变量}}` 都是占位符，使用前替换成实际值。

```
{{股票名称}}  →  五粮液
{{股票代码}}  →  000858
{{核心产品}}  →  普五
{{PE低位}}    →  22
```

完整变量表在 `00_变量速查表.md`，分三类：

| 类型 | 变量 | 说明 |
|------|------|------|
| 必填 | `{{股票名称}}` `{{股票代码}}` `{{核心产品}}` 等 | 每次换股都要改 |
| 估值 | `{{PE低位}}` `{{PE中位}}` `{{PE高位}}` | 根据标的历史数据填 |
| 按需 | `{{具体事件}}` `{{大盘涨幅}}` | 特定问题才用到 |

## 使用流程

### 第一步：选模板

根据你当前所处的交易节点选文件：

| 你在做什么 | 用哪个模板 |
|-----------|-----------|
| 空仓，不知道要不要进 | `01_空仓观望_template.md` |
| 想了解这只票的基本情况 | `02_信息搜集_template.md` |
| 纠结要不要买、买在哪 | `03_分析决策_template.md` |
| 准备建仓，怎么买 | `04_买入建仓_template.md` |
| 已持仓，股价在波动 | `05_持仓波动_template.md` |
| 已盈利，要不要卖 | `06_盈利管理_template.md` |
| 已亏损，怎么处理 | `07_亏损管理_template.md` |
| 到了止损/止盈位 | `08_触发止损止盈_template.md` |
| 准备离场 | `09_离场执行_template.md` |
| 交易结束，复盘 | `10_复盘总结_template.md` |
| 等待下次机会 | `11_等待下一次机会_template.md` |

### 第二步：选问题

每个模板里有 5-8 个问题，找到最接近你当前困惑的那个，复制它下面的提示词块（两个 ` ``` ` 之间的内容）。

### 第三步：替换变量

把 `{{变量}}` 换成实际值。有三种方式：

**方式一：手动替换（最简单）**

粘贴到 AI 对话框后，直接编辑，把 `{{股票名称}}` 改成你的股票名。

**方式二：编辑器查找替换**

在 Cursor / VS Code / Windsurf 里打开模板文件，`Cmd+H`（Mac）或 `Ctrl+H`（Windows）全局替换：

```
查找：{{股票名称}}
替换：五粮液
```

**方式三：命令行批量替换**

换一只新股票时，一条命令生成可用版本：

```bash
sed -e 's/{{股票名称}}/五粮液/g' \
    -e 's/{{股票代码}}/000858/g' \
    -e 's/{{核心产品}}/普五/g' \
    -e 's/{{系列产品}}/低度五粮液、五粮春/g' \
    -e 's/{{行业板块}}/白酒/g' \
    -e 's/{{行业描述}}/高端白酒消费龙头/g' \
    -e 's/{{竞品列表}}/贵州茅台（600519）、泸州老窖（000568）/g' \
    -e 's/{{指数名称}}/沪深300/g' \
    -e 's/{{行业指数}}/中证白酒/g' \
    -e 's/{{PE低位}}/22/g' \
    -e 's/{{PE中位}}/28/g' \
    -e 's/{{PE高位}}/35/g' \
    -e 's/{{批价阈值}}/1000元/g' \
    -e 's/{{估值历史起点}}/2016年1月1日/g' \
    03_分析决策_template.md
```

输出直接复制给 AI 即可。

### 第四步：发给 AI

替换完成后，直接粘贴到任意 AI 工具的对话框发送。

## 实际例子

**场景**：你持有五粮液，今天跌了 3 个点，纠结要不要补仓。

1. 选模板：`05_持仓波动_template.md`
2. 找问题：**问题2｜高频 "跌了3个点了，要不要补仓摊低成本？"**
3. 复制提示词，替换变量：

```
{{股票名称}} → 五粮液
{{股票代码}} → 000858
{{核心产品}} → 普五
{{PE低位}}   → 22
{{PE高位}}   → 35
```

4. 粘贴给 AI，得到结构化分析。

---

## 注意事项

- 每个模板末尾都有**快速填参示例**，以贵州茅台为参考，换股时对照修改
- `{{具体事件}}` 这类变量只在特定问题里出现，用到时再填，不用提前准备
- 模板里的 PE 阈值是茅台的历史数据，换其他股票时一定要重新查该股的历史 PE 区间
