# A股Agent提示词模板 — 变量速查表

> 所有 `_template.md` 文件共用以下变量。使用前全局替换即可。

## 通用变量（每次换股必填）

| 变量 | 说明 | 贵州茅台示例 |
|------|------|-------------|
| `{{股票名称}}` | 股票中文名 | 贵州茅台 |
| `{{股票代码}}` | 交易所代码 | 600519 / 600519.SH |
| `{{行业板块}}` | 所属行业 | 白酒 |
| `{{申万分类}}` | 申万行业分类 | 食品饮料（一级）/ 白酒（二级）|
| `{{核心产品}}` | 公司核心产品 | 飞天茅台 |
| `{{系列产品}}` | 次级/系列产品 | 茅台1935、王子酒 |
| `{{行业描述}}` | 简短行业定位 | 高端白酒消费龙头 |
| `{{竞品列表}}` | 主要竞争对手 | 五粮液（000858）、泸州老窖（000568）、山西汾酒（600809）|
| `{{指数名称}}` | 参考大盘指数 | 沪深300 |
| `{{行业指数}}` | 行业指数 | 中证白酒 |

## 在 AI Coding 工具中的使用方式

### Cursor / Windsurf / Copilot Chat
1. 打开对应的 `_template.md`
2. 全选内容，复制到对话框
3. 在发送前，用「查找替换」把 `{{变量}}` 换成实际值

### Kiro / Claude Code
直接在对话中粘贴模板内容，变量用实际值替换后发送。

### 批量替换脚本（macOS/Linux）
```bash
# 示例：把模板中的茅台变量替换为五粮液
sed -e 's/{{股票名称}}/五粮液/g' \
    -e 's/{{股票代码}}/000858/g' \
    -e 's/{{核心产品}}/普五/g' \
    -e 's/{{行业板块}}/白酒/g' \
    -e 's/{{PE低位}}/22/g' \
    -e 's/{{PE中位}}/28/g' \
    -e 's/{{PE高位}}/35/g' \
    01_空仓观望_template.md
```
