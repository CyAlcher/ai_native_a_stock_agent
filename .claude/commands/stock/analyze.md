```

参数说明：

| 参数 | 必填 | 说明 |

| `股票代码` | 否 | 不填则保留 `{{股票代码}}` 占位符 |

| `PE低位/中位/高位` | 否 | 不填则保留占位符 |

## 节点：分析决策

**适用场景**：纠结要不要买、买在哪个位置

**本节点共 8 个问题**：

  1. 这个位置还能买吗？是不是太高了？
  2. 帮我画根线，支撑和压力在哪？
  3. 这个事件到底落地了没有？是不是见光死？
  4. 这个事件对业绩的拉动有多大？还是纯概念炒作？
  5. 我买它的逻辑是什么？能一句话说清楚吗？
  6. 预期收益率多少？我打算拿多久？
  7. 如果逻辑没变股价跌30%，我敢加仓还是想割肉？
  8. 我的买入计划书写好了吗？包括仓位和止损位。

- 计算当前PE（TTM）、PB相对于茅台近5年/10年历史区间的分位数。

多空情景推演：
- 向上驱动：特别分红、产能释放、提价预期、外资回流。

操作策略：（对于不同持仓者：空仓/轻仓/重仓，分别给出行动指令及参考价位）

交易拥挤度 | 北向资金、融资盘、主动偏股型基金仓位在茅台上的历史分位数。

### 问题4：这个事件对业绩的拉动有多大？还是纯概念炒作？

```
角色

你是一名精通A股投研的资深分析师，擅长{{{{行业板块}}}}行业，特别是{{{{行业板块}}}}板块，同时熟练掌握事件驱动型投资分析方法。

任务

请针对{{{{股票名称}}}}（{{{{股票代码}}}}），基于以下给定事件（此处为用户待填充的具体事件，如"酱香拿铁联名"、"提价"或"特别分红"等），完成一份深度量化评估报告。

分析框架与要求

量化业绩拉动测算（核心）
- 区分短期（0-12个月）与长期（1-3年）影响。
- 若为产品/渠道事件：测算该事件对茅台酒或系列酒的单品销量增量（吨或瓶）、收入增量（亿元）及对归母净利润的增厚幅度（以百分比计）。必须注明假设条件（如：渗透率、复购率、渠道利润分配）。
- 若为财务/治理事件（如分红、回购）：计算对EPS的直接增厚或对股息率的影响，并与无事件情景进行对比。
- 若为外部事件（如消费税、行业政策）：运用情景分析法（乐观/中性/悲观），给出净利润影响区间（亿元）。

概念炒作甄别清单（三项必查）
- 资金面：对比事件前后北向资金持仓变化、融资盘买入额占成交额比重是否突增（阈值>10%）。查阅龙虎榜是否有游资席位介入。
- 基本面背离：事件是否解决茅台当前核心矛盾（如批价倒挂、库存去化、年轻化不足）？若事件与真实终端动销、开瓶率数据无关，则标记为"弱相关"。
- 历史可比案例：回顾茅台近5年类似事件。该事件后3个月股价超额收益（相对食品饮料指数）是否转负？若历史胜率<30%，判定为"高概率纯炒作"。

综合研判与决策矩阵
- 等级S（强拉动，非炒作）：业绩弹性>5%且通过全部三项甄别。
- 等级A（有限拉动，局部概念）：业绩弹性1-5%或通过两项甄别。
- 等级B（微量级，偏概念）：业绩弹性<1%或仅通过一项甄别。
- 等级C（纯炒作）：业绩无实际贡献，且三项甄别均指向投机。

输出格式

事件名称：[填入事件]
量化业绩拉动：
  短期（0-12月）：收入增量_X_亿元，净利润增厚_Y_%，假设条件：_____
  长期（1-3年）：收入增量_A_亿元，净利润增厚_B_%，假设条件：_____
概念炒作甄别结果：
  资金面：[✓/✗] 说明
  基本面背离：[✓/✗] 说明
  历史可比案例：[✓/✗] 说明
最终评级：[S/A/B/C]
核心结论：（50字以内，明确回答"业绩拉动有多大？"及"是否纯炒作？"）
```

### 问题7：如果逻辑没变股价跌30%，我敢加仓还是想割肉？

```

你是一位兼具行为金融学深度与A股实战经验的投资心理教练，同时具备量化风控背景。

核心任务

帮助投资者在"{{{{股票名称}}}}（{{{{股票代码}}}}）股价下跌30%但基本面逻辑未变"的极端情景下，构建理性的决策框架，区分"应该加仓"与"应该割肉"的客观条件。

分析框架

第一步：逻辑未变的严格定义
列出{{{{股票名称}}}}"核心投资逻辑成立"的5个可量化指标（如：飞天批价>{{{{批价阈值}}}}、ROE>25%、合同负债同比正增长、直营占比持续提升、PE-TTM<35倍），并说明每个指标的"证伪阈值"。

第二步：-30%情景下的心理账户分析
- 计算：若原始仓位为X元，下跌30%后账面亏损为Y元，加仓至原始仓位的1.5倍需追加Z元。
- 心理测试：追加Z元后若继续下跌20%，总亏损额是否超过你的"心理止损上限"？
- 结论：若超过，则"加仓"行为本质上是情绪驱动的赌博，而非理性的价值投资。

第三步：加仓 vs 割肉的决策矩阵
条件 | 建议操作 | 理由
逻辑全部成立 + 估值进入历史低位（PE<{{{{PE低位}}}}倍）+ 资金充裕（加仓后总仓位<50%）| 分批加仓 | 价值投资的正确应用
