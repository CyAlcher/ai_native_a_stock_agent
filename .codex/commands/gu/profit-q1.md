分析框架

1. "落袋为安"冲动的心理根源
- 处置效应（Disposition Effect）：投资者倾向于过早卖出盈利的股票，过晚卖出亏损的股票。
- 统计真相：若茅台的投资逻辑未变，过早止盈5%可能错过后续20%-50%的涨幅。
- 核心问题：止盈的依据应该是"逻辑是否兑现"，而非"涨了多少"。

2. 止盈决策的三个维度
- 维度1：估值是否已到高位？
  若当前PE-TTM>40倍（历史高位）→ 可以考虑止盈。
  若当前PE-TTM<35倍（历史中位）→ 估值未到高位，不建议止盈。

- 维度2：投资逻辑是否已经兑现？
  若买入逻辑是"批价回升"，而批价已经回升至目标位 → 逻辑兑现，可以止盈。
  若买入逻辑是"长期持有享受复利"，5%涨幅远未达到目标 → 不建议止盈。

- 维度3：是否有更好的资金去向？
  若有明确的、估值更低的优质标的 → 可以考虑换仓。
  若没有明确的替代标的 → 止盈后资金闲置，机会成本较高。

3. 分档止盈方案（推荐）

- 方案C（全部止盈）：若估值已到历史高位且逻辑完全兑现 → 全部止盈，等待回调后重新布局。

4. 止盈后的纪律
- 止盈后不得立即追涨买回（至少等待5个交易日）。
- 若止盈后股价继续上涨超过5%，不得因懊悔而追涨。
