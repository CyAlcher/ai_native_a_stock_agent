## 写在前面

做股票这几年，最折磨人的从来不是行情，而是那些反复出现的"小纠结"：

- 跌了 3 个点，要不要补仓摊成本？
- 冲高没卖，回落了还能上去吗？
- 刚割完就涨停，要不要追回来？
- 空仓好难受，是不是该进场了？

这些问题在交易群里问，十个人十个答案；问 AI，它又经常跑题，讲一堆技术指标定义，不回答我真正关心的那一下。

所以我花了一段时间，把散户交易全周期拆成 **11 个决策节点、83 个高频问题**，每个问题都配一套结构化分析框架，做成了一个开源工具：

**A 股 AI 决策副驾（ai_native_a_stock_agent）**

GitHub 地址放在文末。今天这篇，讲讲它是什么、怎么用、能解决什么问题。

## 它解决的核心问题

**AI 用来炒股的最大痛点不是"不够聪明"，而是"不懂你在问什么"。**

同样一句"这个位置能买吗"，在空仓踏空时、在小幅盈利时、在深度套牢时，该给的回答完全不一样。

通用 AI 不知道你处于哪个节点，就只能给一套万能套话。

这个工具做的事情很简单：**把散户的心理阶段先分类，再让 AI 按对应的框架回答。**

## 怎么用

### 方式一：命令行工具（推荐）

支持四种主流 AI 开发工具，任选一个：

```
Claude Code / Codex CLI / Gemini CLI / Cursor
```

安装一行命令：

```bash
git clone https://github.com/CyAlcher/ai_native_a_stock_agent.git
cd ai_native_a_stock_agent
bash install.sh
```

脚本会自动检测你本地装了哪个工具。也可以手动指定：

```bash
bash install.sh claude    # Claude Code
bash install.sh codex     # Codex CLI
bash install.sh gemini    # Gemini CLI
bash install.sh cursor    # Cursor
bash install.sh all       # 全部
```

### 方式二：直接提问

装好之后，在工具里直接说人话：

```
统联精密 301168 今天回调了 8%，能建底仓吗？
五粮液 000858 赚了 5 个点，要不要止盈？
贵州茅台 600519 破了 20 日线，要出来吗？
```

AI 会自动识别你处于哪个节点、匹配到哪个问题，用对应框架给你结构化分析。

### 方式三：指定节点

如果你想精确触发某个框架：

```
/gu/buy-q1     统联精密 301168 今天回调了 8%
/gu/hold-q2    五粮液 000858 跌了 3 个点
/gu/profit-q1  贵州茅台 600519 赚了 5 个点
```

### 方式四：任意 AI 都能用

没装上面那些工具也不要紧。DeepSeek、Kimi、豆包、ChatGPT 都行：

```bash
# 查看某个问题的框架
cat .claude/commands/gu/buy-q1.md

# 把内容粘贴到任意 AI 对话框，替换股票名称后发送
```

或用脚本生成可直接粘贴版本：

```bash
bash render.sh buy-q1 统联精密 301168
```

## 一个真实的使用例子

问题：`统联精密 301168 今天回调了 8%，能建底仓吗？`

AI 匹配到「买入建仓」节点的 q1 框架，输出结构大致是：

- **当前位置判断**：均线系统、支撑位、相对高低位
- **回调性质**：健康回踩 vs 趋势破坏
- **建底仓的前提条件**：仓位、止损位、分批节奏
- **风险提示**：最大可能亏损、情绪陷阱
- **具体建议**：建 / 观察 / 不建 的三档结论

比起让 AI "随便聊聊"，这种方式能把散户最容易在冲动下做错的那一步，往回拉一下。

## 为什么要做这个

说实话，散户在市场里最大的敌人从来不是庄家，是自己的反射神经：

- 跌了就想割，涨了就想追
- 空仓手痒，满仓焦虑
- 赚一点就怕飞，亏一点就装死

这些瞬间，**需要的不是预测涨跌，而是一个强制降速的结构。**

AI 写不了行情，但能帮你把"我要做什么"先想清楚。
